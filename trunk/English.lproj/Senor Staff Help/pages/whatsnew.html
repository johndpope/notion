What's New in Señor Staff


Version 0.7.2


Audible note placement feedback
The note you're about to place (or the chord it will create) is now sounded as you move your mouse. This can be turned off in the Preferences dialog.

"Play selected" button
Clicking the "Play selected" button will play only the notes you have selected.

New keyboard shortcuts
Pressing "A"-"G" will place the appropriate note at the mouse cursor. (The shortcuts for accidentals have been moved to the right side of the keyboard to free up the "A" key.)

Version 0.7.1


Note selection

Double-clicking in a measure now selects all notes in that measure. Holding SHIFT while double-clicking will select all notes in the measure, plus any notes between the measure and the notes that were previously selected.

When multiple notes are selected, a colored background is placed behind them to make it easier to see.

Bug fixes

Fixed a bug where loading a saved song containing chords would cause undesired behavior.

MIDI channel selectors for each staff are now correctly re-initialized when loading a saved song.

Further improved redraw performance.

Version 0.7


Compound time signatures

Señor Staff now supports compound time signatures, which alternate between two time signatures from measure to measure.

Better drawing of short notes

Where possible, consecutive eighth notes (and shorter) are connected by the appropriate beams.

Help system

Documentation has been added under the "Señor Staff Help" menu item.

Bug fixes

Fixed some issues with the playback head, particularly with regards to repeating measures.

Improved redraw performance.

Fixed some note spacing issues.

Deletion of key signatures now works.

C major key signature is displayed with naturals when it comes after another key signature - previously it was invisible.
