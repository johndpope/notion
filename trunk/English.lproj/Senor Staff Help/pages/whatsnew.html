What's New in Señor Staff


Version 0.8


Real-time command help
Move your mouse over any part of your score, and the real-time command help window will show you what commands are available.

Bug fixes

Notes were being played an octave too low; they're now played at the appropriate pitch. Old save files will maintain their pitch and appear lower on the staff.

Selection of external MIDI devices from the MIDI menu is now functional.

Transposition of notes with accidentals when changing key signatures has been improved.

Dragging a tied note to change its pitch now breaks ties with other notes when appropriate.

Chords can now be dragged up and down in the same way as single notes.

Fixed some ligature drawing bugs.

Fixed a bug where notes in a chord were not responsive to key commands when Option was held down.

Measure boundaries are now properly recalculated after adding or removing a dot to an existing note.

Version 0.7.5


New staff controls

Each staff in your song now has a name, editable via the text box to its left.

A staff can now be transposed up or down by up to 3 measures by setting the number of half-steps in the transposition control.

MIDI export
Any song can now be exported to a MIDI file using the "Export As..." command in the File menu. Due to a bug in Apple's CoreMidi library, MIDI export does not yet work on Intel Macs. To export to MIDI on an Intel Mac, quit Señor Staff, force it to run with Rosetta, and do the export.

Audible note placement feedback
The note you're about to place (or the chord it will create) is now sounded as you move your mouse. This can be turned off in the Preferences dialog.

"Play selected" button
Clicking the "Play selected" button will play only the notes you have selected.

Automatic transposition
When changing the key signature of a measure, you can now choose to transpose all affected measures accordingly. (This feature is experimental, and may not correctly transpose notes with accidentals.)

New keyboard shortcuts
Pressing "A"-"G" will place the appropriate note at the mouse cursor. (The shortcuts for accidentals have been moved to the right side of the keyboard to free up the "A" key.)

Zoom slider
You can use the new slider in the lower-right corner of the window to zoom in and out, in case you need to see more of your song at a glance.

Bug fixes

Fixed a bug where the "paste" command would not respond under certain circumstances.

Fixed a bug where after loading a file from disk, undo and save would not work.

Eliminated unpredictable behavior after selecting multiple notes and deleting them.

Notes now line up correctly when parallel measures have different key signatures.

Changing a staff to a drum staff when it has notes it can't translate no longer hoses the app.

Version 0.7.1


Note selection

Double-clicking in a measure now selects all notes in that measure. Holding SHIFT while double-clicking will select all notes in the measure, plus any notes between the measure and the notes that were previously selected.

When multiple notes are selected, a colored background is placed behind them to make it easier to see.

Bug fixes

Fixed a bug where loading a saved song containing chords would cause undesired behavior.

MIDI channel selectors for each staff are now correctly re-initialized when loading a saved song.

Further improved redraw performance.

Version 0.7


Compound time signatures

Señor Staff now supports compound time signatures, which alternate between two time signatures from measure to measure.

Better drawing of short notes

Where possible, consecutive eighth notes (and shorter) are connected by the appropriate beams.

Help system

Documentation has been added under the "Señor Staff Help" menu item.

Bug fixes

Fixed some issues with the playback head, particularly with regards to repeating measures.

Improved redraw performance.

Fixed some note spacing issues.

Deletion of key signatures now works.

C major key signature is displayed with naturals when it comes after another key signature - previously it was invisible.
