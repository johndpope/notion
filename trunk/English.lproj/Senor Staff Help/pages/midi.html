Controlling MIDI playback and MIDI files

To change the MIDI playback device:

  1. Open the MIDI menu in the app's menu bar.
  2. Click on the desired MIDI playback device.
  3. If the MIDI playback device you're looking for is not in the menu, quit Señor Staff, run the "Audio MIDI Setup" utility provided with OS X, make the necessary adjustments, and restart Señor Staff.

To set the MIDI channel for a staff:

  1. Select the desired channel from the staff's channel selection menu to the left of the editing window.

MIDI channel 10 is always used for drum staffs. Normal staffs can't be assigned to channel 10, and drum staffs can't be assigned to any other channel.

To control which staffs are played:

  1. Click the "M" button to the left of the editing window to mute a staff (prevent it from being played).
  2. Click the "S" button to the left of the editing window to solo a staff (prevent all other staffs from being played).

To import a song from a MIDI file:

  1. Select "Open..." from the File menu.
  2. Open the MIDI file as you would any other file.

To export a song to a MIDI file:

  1. Select "Export as..." from the File menu.
  2. Select "MIDI file" from the "file format" dropdown.

To export a song to a MusicXML file:

  1. Select "Export as..." from the File menu.
  2. Select "MusicXML file" from the "file format" dropdown.

To export a song to a LilyPond file:

  1. Select "Export as..." from the File menu.
  2. Select "LilyPond file" from the "file format" dropdown.
